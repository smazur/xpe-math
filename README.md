# xpe-math
Simple math expression parser based on [XPE](https://github.com/smazur/xpe) library.

Following features are supported:
* Binary operators: '+', '-', '\*', '/', '%', '^'
* Unary operators: '+', '-'
* Expressions in parentheses: '(2 + 2) * 2'
* Variables
* Functions

## REPL
Run `repl.php` to play.
